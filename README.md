# tukevejtso

Tukevejtso means discipline in Bora.

This project is a local operations toolkit: small terminal commands for keeping a workstation useful, inspectable, and ready for long-running work.

## Platforms

- [Windows](windows/README.md)

## Current Focus

The Windows version starts with the tools we already use locally:

- Docker container helpers.
- A Windows Update reboot guard for long trainings and Codex sessions.
- Small security and terminal diagnostics.
- A dependency-light terminal rendering layer inspired by the `iinuji` primitives.

The intent is to keep the core practical, then make the terminal experience polished enough to open source cleanly.
